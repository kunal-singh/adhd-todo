# to-do

Most to-do apps make ADHD worse. Long lists are the problem, not the solution. This one shows two things at a time: your current task and the next one.

Add a task — name, start time, duration. Mark it done when you're finished and the next one takes its place. There's a search modal for finding old tasks. Settings has one toggle: whether overlapping tasks are allowed.

React 19, TypeScript, Vite, Tailwind. All data stays in IndexedDB. No backend, no account.

```bash
pnpm install
pnpm dev
```

```bash
pnpm build    # outputs to dist/
pnpm preview
```
